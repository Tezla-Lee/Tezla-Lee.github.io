github과 git의 연동…?

My first-repo

  • git init 을 하기 전에 현재 위치를 꼭 확인한다.
  • 중간중간 git status를 사용하여 확인해주면 더 좋다.

Second push to my first-repo

  • git add {file name}으로 원하는 파일만 stage(index)로 올릴 수 있다.
  • git add * 또는 git add . 은 모든 파일(수정된 파일)을 stage로 올린다.

Commit convention

  • 커밋 제목은 50자 이내로 요약하여 작성한다.
  • 제목과 내용사이 한 칸
  • prefix를 이용하여 한 눈에 커밋의 용도를 쉽게 한다.

Start Project with git clone

  • 디렉토리를 주의하여 git clone 한다.

Github을 이용해 Page 만들기

  • github 저장소를 활용해 정적인 사이트 호스팅 가능 (1개 제한)
  • user.name.github.io로 repo 작성하면 끝
